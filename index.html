Gateway BOXIO MQTT 2.0.0
  • application/json

This is the documentation for MQTT communication from the BoxIO point of view. It describes how it interacts with the server. In particular, there are three type of requests;

  1. Requests (xx/request); request to receive some kind of information from the gateway
  2. Updates (xx/update); recurring update sent from the gateway to the broker
  3. Responses (xx/response); response to a request.
