root automatism complex_rules ComplexRule default_enable
Type: boolean

The initial enabled state of the rule.

root automatism complex_rules ComplexRule interval
Type: object

Defines an interval in which the rule is automatically enabled. Inside this daily interval the rule can be enabled or disabled. Outside the interval the rule will always be disabled.

root automatism complex_rules ComplexRule user_operation
Type: enum (of string) Default: "allowed"

Defines the behaviour used when a user operation is requested on parameters/endpoints currently controlled by this rule when it is enabled. 'allowed'/'blocked' permits/blocks the execution of user operations (only on specific parameters or entire endpoints), 'disable_rule' disables automatically the rule when a user operation is executed.

Must be one of:

  • "allowed"
  • "blocked_endpoint"
  • "blocked_parameter"
  • "disables_rule"

root automatism complex_rules ComplexRule user_can_change_enable_state
Type: boolean Default: true

Defines if users can enable/disable this complex rule.

Type: string

The time zone that is taken into consideration when evaluating all time related fields of the automatisms. If not present UTC will be used. It must be a valid iana id (https://www.iana.org/time-zones).
